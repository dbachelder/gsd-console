<task type="checkpoint:human-action" gate="blocking">
  <what>GitHub repository rename and npm publish</what>
  <why>Repository rename requires GitHub web UI, npm publish requires authentication</why>
  <instructions>
    **Step 1: Rename GitHub repository**

    1. Go to https://github.com/dnakov/gsd-tui/settings
    2. In "Repository name" field, change "gsd-tui" to "gsd-console"
    3. Click "Rename"
    4. GitHub will create automatic redirects from the old URL

    **Step 2: Update local git remote**

    After renaming on GitHub, update your local clone:
    ```bash
    git remote set-url origin https://github.com/dnakov/gsd-console.git
    git remote -v  # Verify new URL
    ```

    **Step 3: Push tags and commits**

    ```bash
    git push origin main
    git push origin v1.0.0
    ```

    **Step 4: Publish to npm**

    If you haven't logged into npm:
    ```bash
    npm login
    ```

    Then publish:
    ```bash
    bun publish --access public
    ```

    If 2FA enabled, use:
    ```bash
    bun publish --access public --auth-type web
    ```

    **Step 5: Verify publication**

    ```bash
    # Wait a minute, then verify
    npm view gsd-console
    # Or visit https://www.npmjs.com/package/gsd-console
    ```
  </instructions>
  <resume-signal>Type "published" when npm publish succeeds, or describe any issues</resume-signal>
</task>
